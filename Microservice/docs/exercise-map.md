## 0. Sample Microservices exercises.pdf

- Exercise 2, Authorization Server and Resource Server setup: `security-lab`.
- Exercise 3, JSON Web Tokens for secure communication: `security-lab`.

## 0. Sample Microservices Load balancing exercises.pdf

- Exercise 1, Edge service routing and filtering: `api-gateway` with `GatewayRoutes` and `LoggingFilter`.
- Exercise 2, Load balancing in an API Gateway: `api-gateway` uses `lb://` routes and Spring Cloud LoadBalancer.
- Exercise 3, Resilience patterns in an API Gateway: `api-gateway` has a circuit breaker route and fallback controller.

## 1. Microservices using Spring Boot 3 exercises.pdf

- Build a User and Order Management System: `user-service` and `order-service`.
- Inventory Management with Service Discovery: `product-service`, `inventory-service`, `discovery-server`, and `config-server`.
- Implement an API Gateway: `api-gateway`, `customer-service`, and `billing-service`.
- Resilient Microservices with Circuit Breaker: `payment-service`.

## 2. Microservices with API gateway.pdf

- Account and loan microservices: `account-service` and `loan-service`.
- Eureka discovery server and microservice registration: `discovery-server`, plus Eureka clients in the services.
- Greet service through API gateway: `greet-service` and `api-gateway`.
- Global request logging filter: `api-gateway` `LoggingFilter`.
